Mengenal Semantic Elements Di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel artikel yang ada di webstie kita, tag article bisa kita gunakan dengan element header dan footer.

    Karena tag article memiliki type bloc, artinya element ini memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas penulis artikel

    Karena aside memiliki type sifat sifat block maka lebarnya sesuai page kita tidak menentukan lebarnya.
